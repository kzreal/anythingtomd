## 代码修复记录

在测试过程中发现并修复了以下问题：

### 问题: Excel 预览时 file_count 显示为 0

**原因:** `ExcelConverter.get_preview_text()` 方法没有设置 `self.sections`，导致 API 返回的 `file_count` 为 0。

**修复位置:** `converters/excel_converter.py:90-109`

**修复内容:** 在 `get_preview_text()` 方法中添加了设置 `self.sections` 的代码，使文件计数正确显示。
